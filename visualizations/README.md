# 3D Algorithm Visualizations

Interactive 3D visualizations of metaheuristic algorithms using Plotly.

## Features:
1. PSO Particle Swarm Animation - particles moving in 3D search space
2. GWO Wolf Pack Visualization - alpha, beta, delta wolves hunting
3. Interactive Parameter Exploration - adjust parameters in real-time

## Usage:
Run in Jupyter/Colab or add to Streamlit dashboard.
