# Nature-Inspired Computation for Deep Learning

Welcome to this interactive book on using **Nature-Inspired Computation** to optimize deep learning models!

## What You'll Learn

- How metaheuristic algorithms work (PSO, GWO, WOA, etc.)
- How to apply them to hyperparameter optimization
- How to combine them with Explainable AI (XAI)
- Practical implementation in Python

## Project Overview

| Component | Details |
|-----------|---------|
| **Dataset** | IMDB Movie Reviews (25K+ samples) |
| **Model** | Bidirectional LSTM |
| **Algorithms** | 11 unique metaheuristics |
| **XAI Methods** | SHAP, LIME, Grad-CAM |
