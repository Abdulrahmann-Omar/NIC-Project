üê¶ PSO: Particle Swarm Optimization

How it works:

  • Each particle = set of hyperparameters
  • Particles share best positions
  • Balance: personal vs global knowledge

velocity = w * velocity +
    c1 * r1 * (p_best - pos) +
    c2 * r2 * (g_best - pos)
position += velocity

Inventors:
Kennedy & Eberhart (1995)
